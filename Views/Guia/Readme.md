# GUIAS DE USO EN VUEPRESS

## INICIO DE REPOSITORIO

<!-- Acciones -->

#### Acciones

- Creamos una nueva carpeta
- Inicializamos y configuramos Package.json
- Creamos y adaptamos .gitignore
- Inicializamos Git

- Server
- Opcion de Produccion para VuePress
- Commit

<!-- Acciones -->

<!-- Command -->

#### Command

```
npm install -g vuepress
mkdir VuePress
cd VuePress
npm init -y

git init
git add .
git commit -m "Inicio de Repositorio"
```

<!-- Command -->

## INSTALACION DE VUEPRESS

<!-- Acciones -->

#### Acciones

- Instalamos VuePress
- Ejecutamos Server
- Opcion de Produccion para VuePress
- Commit

<!-- Acciones -->

<!-- Command -->

#### Command

```
npm install -D vuepress

vuepress dev
vuepress build

git add .
git commit -m "Instalacion de VuePress"
```
